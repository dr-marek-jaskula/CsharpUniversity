# .NET Learning Solution

In this solution I present wide range of .NET topics for the learning purpose. 

## Technologies

* C#
* ASP.NET Core 
* Entity Framework Core

#### Main features
* Logging
* Versioning
* Authorization 
* Retries, Timeouts, Cache, Fallbacks
* Health Checks
* Mapping
* Customized Swagger
* Validation
* Code First 
* Custom Migrations
* Query optimization
* Random data generating
